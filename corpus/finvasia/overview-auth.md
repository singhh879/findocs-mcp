# Finvasia Shoonya API — Overview and Authentication

Finvasia's Shoonya API is a free trading API offering REST endpoints for orders,
positions, and account data, plus a WebSocket feed for live market data and order
updates. There is no charge for API access.

## Login and 2FA

Authenticate by POSTing to the `/QuickAuth` login endpoint. The request includes the
user id (`uid`), the SHA-256 hash of the password (`pwd`), a 2FA value which is a
TOTP/authenticator code (`factor2`), the `vendor_code`, the `appkey` (the SHA-256 hash
of `uid|api_secret`), and the device `imei`. On success the response returns a session
token (`susertoken`).

## Using the session token

Pass the `susertoken` (the user session token) with each subsequent request, along
with the `uid` and `actid` (account id). The session is valid for the trading day and
must be regenerated by logging in again the next day.

## Request format

Most endpoints accept the payload as a URL-encoded body of the form `jData=...&jKey=...`,
where `jData` is the JSON request and `jKey` is the session token. Responses are JSON
with a `stat` field that is `Ok` on success or `Not_Ok` with an `emsg` error message on
failure.
